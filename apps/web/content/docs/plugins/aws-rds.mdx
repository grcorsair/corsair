## Attack Vectors

### Public Accessibility (`public-accessibility`)

**Severity:** CRITICAL
**MITRE ATT&CK:** T1190 (Exploit Public-Facing Application)

Checks whether the RDS instance is publicly accessible and whether the associated security groups allow inbound traffic from `0.0.0.0/0` on the database port.

### Encryption at Rest Disabled (`encryption-at-rest-disabled`)

**Severity:** HIGH
**MITRE ATT&CK:** T1530 (Data from Cloud Storage Object)

Verifies that storage encryption is enabled using AWS KMS. Unencrypted database storage exposes data if the underlying EBS volumes are compromised.

### Insufficient Backup Retention (`insufficient-backup-retention`)

**Severity:** MEDIUM
**MITRE ATT&CK:** T1485 (Data Destruction)

Checks the automated backup retention period. A retention period of less than 7 days increases the risk of unrecoverable data loss from ransomware or accidental deletion.

### Deletion Protection Disabled (`deletion-protection-disabled`)

**Severity:** MEDIUM
**MITRE ATT&CK:** T1485 (Data Destruction)

Verifies that deletion protection is enabled to prevent accidental or malicious database deletion via the AWS API or console.
