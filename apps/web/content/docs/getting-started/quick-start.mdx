## Prerequisites

- [Bun](https://bun.sh/) runtime (v1.0+)
- An Anthropic API key (for document ingestion via Claude)

## Installation

```bash
# Clone the repository
git clone https://github.com/Arudjreis/corsair.git
cd corsair

# Install dependencies
bun install
```

## Verify an Existing CPOE (No API Key Needed)

The fastest way to see Corsair in action — verify the example CPOE:

```bash
bun run bin/corsair-verify.ts examples/example-cpoe.jwt
```

This verifies the Ed25519 signature, checks expiry, and displays the assurance level and control results.

## Generate a CPOE from a SOC 2 Report

```bash
# Set your Anthropic API key
export ANTHROPIC_API_KEY=your_key_here

# Ingest a SOC 2 report and generate a signed CPOE
bun run corsair.ts ingest --file report.pdf --type soc2
```

This runs the CPOE lifecycle: INGEST → CHART → QUARTER → MARQUE.

## Generate Signing Keys

```bash
# Generate Ed25519 keypair for CPOE signing
bun run corsair.ts keygen --output ./keys
```

## CLI Commands

| Command | Description |
|---------|-------------|
| `corsair ingest` | Ingest a compliance document and generate a signed CPOE |
| `corsair verify` | Verify a CPOE's Ed25519 signature and check validity |
| `corsair keygen` | Generate Ed25519 signing keypair |
| `corsair help` | Show available commands |
