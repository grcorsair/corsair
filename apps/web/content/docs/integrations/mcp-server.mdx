## Overview

Corsair exposes all primitives via the [Model Context Protocol](https://modelcontextprotocol.io/) for integration with Claude Code and other AI agents.

## Setup

Start the MCP server:

```bash
bun run mcp-server.ts
```

Add to your Claude Code configuration (`.claude/mcp.json`):

```json
{
  "mcpServers": {
    "corsair": {
      "command": "bun",
      "args": ["run", "/path/to/corsair/mcp-server.ts"]
    }
  }
}
```

## Example Usage with Claude Code

Once configured, Claude Code can use Corsair tools directly:

```
> Run a Corsair reconnaissance on my S3 bucket "my-app-assets"
```

Claude will call `corsair_recon` with the appropriate parameters and return the results.
